To create Visual Studio 2013 project - run build_Windows.bat
To build on Ubuntu and create Eclipse project - run build_Ubuntu.bat

To run on Windows - copy shaders from the folder /shaders/ into the folder where  exe is.
Folder /thirdparties/ should contain glew-1.13.0, glm and freeglut libraries. 
If you have another version of glew adjust /src/CMakeLists.txt.
